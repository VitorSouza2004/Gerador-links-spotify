# 🎵 Gerador & Checker Spotify

Sistema completo de gerador e checker de links Spotify Premium com autenticação segura.

## ✨ Funcionalidades

- 🔐 **Sistema de Login/Senha** - Acesso restrito com credenciais exclusivas
- ⚙️ **Painel Admin** - Gerenciamento completo de usuários por Discord ID
- 🎲 **Gerador de Links** - Gere links Spotify Premium rapidamente
- 🔍 **Checker de Contas** - Verificação e organização de contas
- 🎮 **Botão Discord** - Acesso rápido ao servidor Discord

## 🚀 Acesso

Acesse o site: [https://vitorsouza2004.github.io/Gerador-links-spotify/](https://vitorsouza2004.github.io/Gerador-links-spotify/)

## 📱 Funcionalidades do Painel Admin

- Adicionar usuários com ID do Discord
- Gerar credenciais automáticas
- Remover acesso de usuários
- Listagem completa de usuários cadastrados
